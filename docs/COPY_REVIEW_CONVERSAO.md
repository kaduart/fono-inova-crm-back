# 📝 Copy Review - Amanda V7: De "Chatbot" para "Vendedora Humana"

## 🎯 Princípios de Conversão Aplicados

1. **Transparência Primeiro** - Preço na frente gera confiança
2. **Menos é Mais** - Remover floreios robóticos
3. **Assunção de Venda** - Assumir que vai agendar, não "se"
4. **Transições Suaves** - Retomar fluxo sem parecer que esqueceu
5. **Validação Afetiva Sutil** - Conexão emocional sem exagero
6. **Escolhas Limitadas** - Técnica de fechamento (2-3 opções, não infinitas)

---

## 📋 Mensagens Revisadas (ANTES → DEPOIS)

### **1. Captura de Nome** 👤

#### ❌ ANTES (robótico, repetitivo)
```
Que nome lindo, João! 🥰

E João tem quantos aninhos?
```

**Problemas:**
- "Que nome lindo" toda vez soa artificial
- Repetir o nome na mesma resposta é padrão de bot
- Emoji 🥰 força simpatia demais

#### ✅ DEPOIS (natural, progressivo)
```
Perfeito, anotei: João 💚

E ele tem quantos anos?
```

**Por quê funciona:**
- "Anotei" = sensação de progresso real (secretária fazendo o trabalho)
- Usa pronome "ele" (não repete nome)
- 💚 emoji neutro-positivo (identidade Fono Inova)
- Menos caracteres = mais objetivo

---

### **2. Confirmação de Idade (Bebê 1-3 anos)** 👶

#### ❌ ANTES (genérico)
```
Perfeito! 🌟

Pra fonoaudiologia, temos ótimos profissionais. Que período funciona melhor: **manhã ou tarde**? 😊
```

**Problemas:**
- "Perfeito!" vazio (sem contexto)
- "Temos ótimos profissionais" = obviedade (toda clínica tem)
- Não aproveita emoção de pai/mãe com bebê

#### ✅ DEPOIS (validação afetiva + assunção)
```
2 anos! Que fase linda pra começar 💚

Qual período funciona melhor: manhã ou tarde?
```

**Por quê funciona:**
- Eco da idade (confirma que ouviu)
- "Fase linda pra começar" = validação + urgência suave (agora é o momento)
- Remove "temos ótimos profissionais" (assumed, não precisa vender competência)
- Pergunta direta sem emojis extras

---

### **3. Confirmação de Idade (Adulto/Adolescente)** 🧑

#### ❌ ANTES
```
Adolescência, né? Uma fase de muitas transformações 💚

Pra atendimento, temos ótimos profissionais. Que período funciona melhor: **manhã ou tarde**? 😊
```

**Problemas:**
- "Fase de muitas transformações" é clichê de psicólogo em filme
- "Pra atendimento" é frio

#### ✅ DEPOIS
```
Entendido! Vamos cuidar bem de você 💚

Qual período funciona melhor: manhã ou tarde?
```

**Por quê funciona:**
- Remove clichê adolescente
- "Vamos cuidar" = compromisso ativo
- Direto ao ponto

---

### **4. Pergunta de Preço** 💰

#### ❌ ANTES (explicação longa, esconde preço)
```
Pra Fonoaudiologia 💬:

Na avaliação, vamos entender exatamente como está a comunicação, identificar pontos fortes e desafios, e traçar um plano personalizado...

É R$ 280 por sessão (50 minutos)

E o melhor: trabalhamos com reembolso de plano! 💚
```

**Problemas:**
- Preço escondido no meio do texto (parece que tem vergonha)
- Explicação antes do valor = desconfiança ("tá tentando justificar que é caro")
- Muito texto antes de responder a pergunta

#### ✅ DEPOIS (transparência gera confiança)
```
A avaliação custa R$ 280 💚

Dura 50 min e já saímos com um planejamento personalizado pra João.

Funciona com reembolso de plano (80-100%)! Qual o nome da criança?
```

**Por quê funciona:**
- Preço na PRIMEIRA LINHA (transparência = confiança)
- Emoji 💚 logo após valor (normaliza, não deixa preço "sozinho e assustador")
- Personaliza com nome da criança (se já tem)
- Retoma qualificação no final (não fica só na objeção de preço)
- Menos caracteres = mais conversão

---

### **5. Interrupção: Plano de Saúde** 🏥

#### ❌ ANTES (genérico, perde contexto)
```
Trabalhamos com reembolso de todos os planos! Você paga e solicita o ressarcimento (geralmente entre 80% e 100%).

Também aceitamos Pix, cartão de crédito e débito! 😊

Quer marcar uma avaliação?
```

**Problemas:**
- Não menciona a operadora específica (usuário perguntou "aceita Unimed?")
- "Quer marcar uma avaliação?" = recomeça do zero (perde contexto)
- Muito genérico

#### ✅ DEPOIS (personalizado, retoma contexto)
```
Sim! Funciona com reembolso da Unimed 💚

Você paga e solicita o ressarcimento (geralmente 80-100%).

Sobre o horário: qual período funciona melhor pra vocês?
```

**Por quê funciona:**
- Confirma a operadora específica ("da Unimed")
- Transição suave: "Sobre o horário:" (não perde o fio)
- Assume que vai agendar ("qual período" não "quer marcar?")
- Remove informação desnecessária (Pix, cartão - isso vem depois)

---

### **6. Exibição de Slots** 📅

#### ❌ ANTES (pergunta sem mostrar opções)
```
Achei essas opções pra Fonoaudiologia 💬:



Qual desses funciona melhor? 😊

(Se nenhum der certo, me avisa que busco outros!)
```

**Problemas:**
- Vazio (sem slots) parece bug
- Pergunta "qual desses?" sem mostrar nada
- Parênteses defensivo ("se nenhum der certo") = assume falha

#### ✅ DEPOIS (quando TEM slots)
```
Encontrei essas opções pra Fonoaudiologia 💬:

A) Terça 14h com Dra. Carla
B) Quarta 10h com Dr. Paulo
C) Quinta 15h com Dra. Ana

Qual serve melhor?
```

**Por quê funciona:**
- Letras (A/B/C) facilitam escolha
- Nome do profissional = credibilidade
- Pergunta direta ("qual serve?")
- Remove parênteses defensivo

#### ✅ DEPOIS (quando NÃO tem slots / erro)
```
Tô vendo que a agenda tá bem movimentada nesse período 💚

Posso te oferecer:
• Quinta 14h
• Sexta 9h

Qual desses funciona?
```

**Por quê funciona:**
- Explica o motivo (agenda cheia) sem parecer erro técnico
- Oferece alternativas concretas (não deixa vazio)
- Técnica de escolha limitada (2 opções = mais fechamento que infinitas)

---

### **7. Coleta de Queixa/Complaint** 💬

#### ❌ ANTES (forçado, muito emoji)
```
Perfeito! 💬

Me conta mais sobre o que tá preocupando. Quero entender direitinho pra poder ajudar!
```

**Problemas:**
- "Quero entender direitinho" = infantilização
- Emoji de therapy depois de "Perfeito!" descontextualizado
- "Pra poder ajudar" é obviedade

#### ✅ DEPOIS (consultivo, profissional)
```
Sobre fonoaudiologia: me conta o que tá acontecendo?
```

**Por quê funciona:**
- Contextualiza ("Sobre fonoaudiologia")
- Pergunta aberta mas direta
- Remove infantilização
- Menos é mais

---

### **8. Saudação Inicial** 👋

#### ❌ ANTES (muito texto de entrada)
```
Oi! Sou a Amanda da Fono Inova! 😊

Que bom que você entrou em contato! Me conta: tá procurando fono, psico, fisio, ou qual especialidade?
```

**Problemas:**
- "Que bom que você entrou em contato!" = formalidade de SAC
- Muito texto antes da pergunta útil

#### ✅ DEPOIS (direto, warm)
```
Oi! Sou a Amanda da Fono Inova 💚

Tá procurando fono, psico, fisio ou qual especialidade?
```

**Por quê funciona:**
- Remove formalidade SAC
- 1 emoji só (💚 identidade)
- Vai direto ao ponto
- Tom casual mantido ("tá procurando")

---

### **9. Objeção: Desistência** 😔

#### ❌ ANTES (exagerado, inseguro)
```
Tudo bem! Sem problemas! 😊

Fico à disposição quando você quiser agendar. Qualquer dúvida, é só me chamar! Estou aqui para ajudar! 💚
```

**Problemas:**
- Muitos pontos de exclamação = insegurança
- "Estou aqui para ajudar!" = obviedade + desperation
- "Quando você quiser" = sem urgência nenhuma

#### ✅ DEPOIS (confiante, porta aberta)
```
Tudo bem! Sem pressão nenhuma 😊

Se mudar de ideia, é só chamar. Estou aqui! 💚
```

**Por quê funciona:**
- "Sem pressão" = validação da decisão (reverse psychology suave)
- Remove excesso de pontos de exclamação
- "Se mudar de ideia" = assume que pode voltar (deixa porta aberta)
- Menos desesperado

---

### **10. Período Recebido (Pré-Busca de Slots)** 📆

#### ❌ ANTES (promessa sem entrega imediata)
```
Perfeito! Anotado manhã! ✅

Agora deixa eu ver os horários...
```

**Problemas:**
- "Deixa eu ver" = delay mental (parece que vai demorar)
- Emoji ✅ + "Anotado" redundante com "Perfeito"

#### ✅ DEPOIS (ação imediata)
```
Manhã anotado! Só um instante enquanto busco os horários... ⏳
```

**Por quê funciona:**
- Remove "Perfeito" genérico
- "Só um instante" = expectativa de rapidez
- ⏳ emoji de loading (sensação de progresso)
- Menos palavras = mais ação

## 🎨 Emojis: Guia de Uso

### **✅ Emojis que FICAM** (identidade + clareza)
- 💚 (identidade Fono Inova, validação suave)
- 💬🧠🏃🤲📚 (ícones de terapias - ajudam escaneabilidade)
- ⏳ (loading, expectativa)

### **❌ Emojis que SAEM** (forçam simpatia)
- 🥰 (amor exagerado)
- 😊 em excesso (todo final de frase fica falso)
- ✅ (redundante com "anotado")
- 🌟 (genérico, sem significado)

### **⚠️ Emojis com MODERAÇÃO**
- 😊 (usar 1x por conversa, não todo turno)
- ✨ (só quando realmente é algo especial)

## 📝 Checklist de Validação (Antes de Deploy)

Antes de mudar cada mensagem, pergunte:

- [ ] Responde a pergunta do usuário na PRIMEIRA LINHA?
- [ ] Tem menos de 3 linhas de texto corrido?
- [ ] Remove obviedades? ("temos ótimos profissionais", "estou aqui pra ajudar")
- [ ] Usa pronomes em vez de repetir nomes?
- [ ] Assume que vai agendar (não "se quiser")?
- [ ] Emoji tem função (identidade/clareza) ou é enfeite?
- [ ] Transição retoma contexto anterior?
- [ ] Tom é "secretária da casa" (não call center)?

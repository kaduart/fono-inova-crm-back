## 🧠 Interpretação do que você quer (fluxo correto)

No card do pacote:

* Pacote é criado
* Ele tem sessões normais
* Mas existe um “gap” → sessões pendentes (ou balance do cliente)
* Você quer um botão tipo:

> **“Importar pendências do cliente”** ou **“Adicionar sessões pendentes”**

Que faz:

* Buscar `patientBalance / pendingSessions`
* Transformar isso em sessões dentro do pacote
* Adicionar automaticamente na lista do pacote (state do front)

---

### 2. Handler (lógica principal)

```ts
const handleImportPendingSessions = () => {
  const pending = patient?.pendingSessions || patient?.balanceSessions || [];

  const mappedSessions = pending.map((session) => ({
    id: crypto.randomUUID(),
    type: "PENDING",
    status: "PENDING",
    date: session.date || null,
    value: session.value || 0,
    source: "BALANCE",
  }));

  setPackageSessions((prev) => [...prev, ...mappedSessions]);
};
```

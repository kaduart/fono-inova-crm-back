# ✅ SOLUÇÃO CORRETA (nível produção)

Você não deve só detectar adulto.

👉 Você precisa classificar em **3 estados**:

```js
PERSONA = {
  SELF: 'self',        // adulto falando de si
  CHILD: 'child',      // responsável falando de criança
  UNKNOWN: 'unknown'
}
```

---

# 🔍 Detector mais robusto

## 1. Detectar criança (ampliado)

```js
const mentionsChild = /(
  filho|filha|
  criança|beb[eê]|
  neto|neta|
  sobrinho|sobrinha|
  afilhado|afilhada|
  aluno|aluna|
  menor|
  pequeno|pequen[oa]
)/i.test(text)
```

👉 Isso resolve seu ponto 💥

---

## 2. Detectar adulto (self)

```js
const isSelf = /(
  para mim|pra mim|
  para eu|pra eu|
  é comigo|
  sou eu
)/i.test(text)
```

---

## 3. Regra final (ESSENCIAL)

```js
if (mentionsChild) {
  stateData.persona = 'child'
  stateData.isAdult = false
} else if (isSelf) {
  stateData.persona = 'self'
  stateData.isAdult = true
} else {
  stateData.persona = 'unknown'
}
```

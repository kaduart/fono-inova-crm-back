# 🚀 Serviço central

Algo tipo:

```js
async function calculateMonthlyProjection(month, year) {

  const appointments = await Appointment.find({
    date: {
      $gte: startOfMonth,
      $lte: endOfMonth
    },

    operationalStatus: {
      $nin: ['cancelled']
    }
  })

  const projectedRevenue =
    appointments.reduce((sum, appt) => {
      return sum + (appt.sessionValue || 0)
    }, 0)

  return {
    projectedRevenue
  }
}
```

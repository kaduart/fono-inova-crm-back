# 🚀 2. CACHE POR MÊS (GANHO ABSURDO)

## 💥 ideia:

Se já buscou abril → não busca de novo

---

## ✅ backend (simples e poderoso)

```javascript
const cache = new Map();

function getCacheKey(start, end) {
  return `${start}_${end}`;
}

app.get('/api/v2/appointments', async (req, res) => {
  const { start, end } = req.query;
  const key = getCacheKey(start, end);

  if (cache.has(key)) {
    return res.json(cache.get(key));
  }

  const data = await Appointment.find({
    date: { $gte: new Date(start), $lte: new Date(end) }
  })
  .select('date time operationalStatus patient doctor')
  .populate('patient', 'fullName')
  .populate('doctor', 'fullName')
  .lean();

  cache.set(key, data);

  // limpa depois de 5 min
  setTimeout(() => cache.delete(key), 300000);

  res.json(data);
});
```

## 🎯 ESTRATÉGIA DA SEQUÊNCIA

### Princípio: Onda de Jornada
```
Semana 1:  Descoberta → Consideração → Decisão
Semana 2:  Descoberta → Retargeting → Consideração
Semana 3:  Decisão → Descoberta → Decisão
Semana 4:  Consideração → Descoberta → Retargeting
Semana 5:  Decisão → Consideração → Descoberta
Semana 6:  Retargeting → Decisão → Consideração
```

**Por que funciona:**
- Não cansa com mesmo estágio seguido
- Cria "caminho natural" para quem acompanha
- Quem vê tudo passa por jornada completa 6x

## ⚡ AJUSTES EM TEMPO REAL

### Se um vídeo "bombar":
→ Replicar estrutura imediatamente
→ Gerar variações 4 e 5 desse tema
→ Boostar com ads

### Se um vídeo "flopar":
→ Não repetir estrutura
→ Analisar: foi timing? tom? CTA?
→ Ajustar nos próximos

### Se WhatsApp explodir:
→ Acelerar para mais decisão
→ Reduzir descoberta temporariamente
→ Aproveitar momentum

## 🚀 APÓS AS 6 SEMANAS

### Se funcionar (lead > 15, agendamento > 5):
→ Gerar variações 4, 5, 6 dos temas vencedores
→ Criar sequência de 12 semanas
→ Escalar com anúncios

### Se não funcionar (lead < 5):
→ Revisar: é o roteiro? A avatar? O CTA?
→ Testar com vídeos reais (profissional gravando)
→ Ajustar tom ou especialidade focada
